Computer Science Classes

Introduction to Computer Science (111)

  • Introduce the student to the fundamental techniques used in computer science and software development
  • By the end of the course the student should be able to design, write, test, and analyze programs to solve simple real-world problems
  • The course covers sections of the following topics: Fundamentals of Programming, Introductory Java Programming, Procedural Programming, Data Structures (Strings, arrays), Efficient Algorithms, Object-Oriented Programming,
